#### 4. 进入容器中默认目录时/TMP

```dockerfile
FROM centos
ENV MYPATH /TMP
WORKDIR $MYPATH
RUN yum -y install vim
RUN yum -y install net-tools
EXPOSE 80
CMD echo "$MYPATH"
CMD echo "success-------ok"
CMD /bin/bash

```
